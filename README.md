Use the auto_x_cancel_helper_linux or the auto_x_cancel_helper to populate the data needed for the main program to work.
To use the helper:
  you must first take a screenshot of your game with a gun out,
  then you must open that screen shot in a full screen mode,
  run the helper and alt+tab back to the full screen screenshot,
  put your mouse tip in the center of the right most bullet and wait at least 10 seconds,
  this should add a config file at /~/.config/auto_x_cancel/bullet-config or at /~/.auto_x_cancel/bullet-config
